**DinoCode** is a programming language based on **programmer intent inference**, designed specifically to facilitate and accelerate software development.

> [!NOTE]
>  Explore the [official documentation](https://dinocode.blassgo.dev) in English and Spanish.

---

## How to try DinoCode

1. **Web Platform**
    
    Visit [dinocode.blassgo.dev](https://dinocode.blassgo.dev) to experiment with the language in your web browser (no installation required). It includes:

    - Complete documentation in English and Spanish
    - Interactive editor with WebAssembly
    - Executable examples
    - Automatically generated flowcharts
    - Real-time bytecode inspection

2. **Command Line**

    Download precompiled binaries from the [Releases](https://github.com/dinocode-lang/dinocode/releases) section

    Run a script:

    ```bash
    dinocode programa.dino
    ```

    Debug options:

    ```bash
    dinocode # Shows help
    dinocode --tokens programa.dino      # Shows generated tokens
    dinocode --bytecode programa.dino    # Shows generated bytecode
    dinocode --tokens --bytecode programa.dino    # Shows generated tokens and bytecode
    ```

## Compile from source

- **Windows:**

    ```bash
    cargo build -p dinocode-cli --release --target x86_64-pc-windows-msvc
    ```

- **Linux** (with zigbuild):

    ```bash
    cargo zigbuild -p dinocode-cli --release --target x86_64-unknown-linux-gnu
    ```
- **With Logging:**

    You can use the `logging` feature flag to inspect the interpreter bootstrap process (you should not use it in production releases):

    ```bash
    cargo build -p dinocode-cli --features logging
    ```

### Syntax example

```dinocode

:add a b
    return a + b

:main args

    > BANNER
     ------------------
     Welcome to DinoCode 🦖
     ------------------

    print BANNER

    x = 10
    y = 1.5
    print x * y
    
    # Both ways work
    print add(x y)
    print $(add x y)

    # Ranges
    for i in 0..100_000
       print i

    print "Command line arguments:"
    for arg in args
       print arg

```
